# create-epics

## Overview
As a Product Manager, decompose project requirements into prioritized, actionable epics with clear business value alignment and requirement traceability. This workflow analyzes functional, non-functional, technical, data, and UX requirements to produce a structured epic backlog.

## Execution
Call MCP tool:
    - ReadPrompt(name="create-epics", version="latest")

- Update ToDo list derived from the returned prompt instructions by readjusting the items.
