# pull-request

## Overview
As a Senior Software Engineer, facilitate safe and well-documented pull request creation with comprehensive validation and user control. This workflow performs repository analysis, change summarization, and conflict detection to produce structured, reviewer-friendly pull requests.

## Execution
Call MCP tool:
    - ReadPrompt(name="pull-request", version="latest")

- Update ToDo list derived from the returned prompt instructions by readjusting the items.
