# review-edge-cases

## Overview
As a Senior QA Engineer, systematically analyze a specified artifact to surface boundary conditions, failure modes, and overlooked edge cases across input, state, concurrency, and integration dimensions. This workflow produces severity-ranked edge case findings with clear reproduction context for downstream remediation.

## Execution
Call MCP tool:
    - ReadPrompt(name="review-edge-cases", version="latest")

- Update ToDo list derived from the returned prompt instructions by readjusting the items.
