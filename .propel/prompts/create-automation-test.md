# create-automation-test

## Overview
As a Senior QA Engineer, generate test workflows from functional requirements by decomposing use cases into happy path, edge case, and error scenarios. This workflow produces structured, automatable test cases with clear pass/fail criteria.

## Execution
Call MCP tool:
    - ReadPrompt(name="create-automation-test", version="latest")

- Update ToDo list derived from the returned prompt instructions by readjusting the items.
