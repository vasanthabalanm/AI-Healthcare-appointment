# create-project-plan

## Overview
As a Project Manager, generate a comprehensive project plan covering scope definition, cost baseline, resource planning, risk management, and governance. This workflow analyzes requirements, architecture, and technical complexity to produce realistic estimates with full requirement traceability.

## Execution
Call MCP tool:
    - ReadPrompt(name="create-project-plan", version="latest")

- Update ToDo list derived from the returned prompt instructions by readjusting the items.
