# review-ux

## Overview
As a Senior UX Designer, perform a heuristic review of UI source files applying Krug's laws, Nielsen's 10 heuristics, and dark-pattern detection to identify usability issues and their impact on users. This workflow produces severity-ranked findings scoped by heuristic category and optional persona journey.

## Execution
Call MCP tool:
    - ReadPrompt(name="review-ux", version="latest")

- Update ToDo list derived from the returned prompt instructions by readjusting the items.
