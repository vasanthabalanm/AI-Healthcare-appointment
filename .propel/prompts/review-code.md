# review-code

## Overview
As a Senior Software Engineer, conduct structured code reviews covering security vulnerabilities, performance issues, architectural compliance, and code quality. This workflow analyzes local changes with technology-specific best practices and produces actionable feedback across all application layers.

## Execution
Call MCP tool:
    - ReadPrompt(name="review-code", version="latest")

- Update ToDo list derived from the returned prompt instructions by readjusting the items.
